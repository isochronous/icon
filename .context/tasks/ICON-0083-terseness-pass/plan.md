## Decisions
- **Reliability is the guardrail, not word count.** This repo's value is precise, reliable agent instructions; the retrospectives are full of "this exact wording / exact-string / fail-closed phrasing is load-bearing" lessons. Terseness may cut redundancy, filler, and repetition — it must NOT strip reliability-load-bearing precision (anti-rationalization tables, fail-closed gates, exact-match strings, gate/hook contracts, security invariants, the verbatim common-constraints block's meaning).
- **ADR-008 alignment**: terseness on always-loaded content (manager/PM roles, the ×9-inlined `shared/common-constraints.md`) directly benefits the token budget — highest-value, measurable via word-count deltas. Savings in the common-constraints block multiply ×9.
- **USER SCOPING DECISION**: **WHOLE TREE** (agents + skills + commands + `.context/` + `context_template/`).
- **CALIBRATION CORRECTION (user feedback, 2026-07-18)**: the first "JPEG-95 / keep tables verbatim" pass was **TOO CONSERVATIVE** (manager only −2%). User: "another agent reduced manager by **15-20%** without negatively impacting behavior." → **The guardrail is BEHAVIOR/RELIABILITY preservation, NOT verbatim preservation of tables/contract prose.** Terse HARD (target ~15-20% on verbose files): compress wording everywhere incl. inside anti-rationalization tables (shorter cells), Hardcoded lists, contract prose, warmstart template, narrative → directives; merge redundant restatements; cut hedging/meta/over-explanation. **PRESERVE:** every distinct rule/trigger/gate/contract and the MEANING of every rationalization row + Hardcoded item (an agent must still be bound by exactly the same behaviors). **TRULY VERBATIM (literal tokens only, can't reword):** hook/gate/script string-matched strings — `<!-- BEGIN/END: common-constraints -->`, marker syntax (`context-graph:orphan-ok`, `pre-commit:dead-ref-ok-*`), `inject-manager-role`, `Phase-Handoff:` trailer, script-parity byte-identity, iconrc keys, cited `.context/<path>` paths, `## Related`/`**Supersedes**`/`**Superseded-by**`/`**Status**` seam tokens, JSON entrypoint field names.
- Approach: front-load an inventory + per-category do-not-trim guardrail (ICON-0080 lesson), then terse in DISJOINT file-group batches (parallel in one worktree), each carrying the guardrail + JPEG-95 calibration + before/after word-count deltas, followed by a reliability-focused @reviewer pass per batch (did any concision change a behavior/contract/exact-string?).
